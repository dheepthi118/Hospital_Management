# Hospital_Management
The Hospital Management System is a frontend-based web application developed using HTML, CSS, and JavaScript. It simulates essential hospital operations through an interactive and easy-to-use interface and is primarily intended for educational and learning purposes.

Key Highlights

Patient Registration

Allows patients to register by entering personal details such as name, age, email, and contact number

Includes basic client-side form validation to ensure required fields are filled correctly

Doctor Availability Management

Displays a list of doctors along with their specializations and available timings

Provides a dynamic search feature to filter doctors by specialization or keywords

Appointment Booking

Enables users to select a preferred date and time for appointments

Demonstrates basic scheduling functionality at the frontend level

Hospital Services Overview

Showcases major hospital services including Emergency, ICU, Pharmacy, and Laboratory Tests

Organized using a responsive card-based layout for better readability

Medicine Inventory Display

Lists available medicines along with their current quantities

Represents how inventory information can be presented in a hospital system

Staff Login Module

Includes a simple login system for staff access

Uses client-side credential validation for demonstration purposes

Patient Feedback System

Features an interactive star-based rating system

Allows patients to provide feedback on their experience visually

Contact and Location

Integrates Google Maps to help users locate the hospital

Provides basic contact visibility through an embedded map view

This project demonstrates fundamental web development concepts such as form handling, DOM manipulation, user interaction, and UI design. It can be further enhanced by integrating backend services, databases, secure authentication, and real-time hospital management features.
